### 🔒 Security Features Added

#### Content Security Policy (CSP)
- **File**: `src/layouts/Layout.astro`
- **Changes**: Added comprehensive CSP meta tag to prevent XSS attacks
- **Policy**: Restricts resource loading to trusted sources only
- **Impact**: Prevents malicious script injection and unauthorized resource loading

#### Security Headers Configuration
- **Files**: `src/layouts/Layout.astro`, `astro.config.mjs`
- **Headers Added**:
  - `X-Frame-Options: DENY` - Prevents clickjacking attacks
  - `X-Content-Type-Options: nosniff` - Prevents MIME type sniffing
  - `Referrer-Policy: strict-origin-when-cross-origin` - Controls referrer information
  - `Permissions-Policy` - Disables dangerous browser features (camera, microphone, geolocation, payment)
- **Implementation**: Dual configuration in HTML meta tags and Vite server headers
