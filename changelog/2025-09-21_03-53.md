# Changelog - 2025-09-21 03:53 (Europe/Helsinki)

## WorkingCommittee Component Redesign

### Changes Made

- **Layout Transformation**: Completely redesigned the `WorkingCommittee.astro` component from a vertical card grid layout to a horizontal profile layout

- **Profile Photo Updates**:
  - Changed from square aspect-ratio images to circular profile photos (96x96px)
  - Photos now positioned on the left side of each profile entry

- **Responsive Layout**:
  - Implemented flexbox with `flex-wrap` for responsive behavior
  - Added minimum width constraints (`min-w-xs`) and maximum width (`max-w-sm`) for optimal sizing
  - Increased gap between items to `gap-10` for better spacing

- **Typography & Styling**:
  - Updated name styling to use `text-gray-900` color with bold font weight
  - Maintained role styling with large, bold text
  - Kept bio text in smaller, readable gray color
  - Preserved LinkedIn icon integration with proper accessibility features

### Technical Details

- **File Modified**: `src/components/WorkingCommittee.astro`
- **Layout Method**: Flexbox with wrap functionality
- **Responsive Breakpoints**: Maintains consistent sizing across all viewport sizes
- **Accessibility**: Preserved all existing accessibility features including ARIA labels and screen reader text

### Visual Impact

The component now displays member profiles in a horizontal card format with circular profile photos, providing a more modern and space-efficient layout that scales well across different screen sizes while maintaining readability and professional appearance.
