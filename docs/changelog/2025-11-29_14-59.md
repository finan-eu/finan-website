# Changelog - 2025-11-29 14:59 (Europe/Helsinki)

## Added

### SEO Infrastructure

- **Installed `@astrojs/sitemap` plugin (v3.6.0)**
  - Automatically generates XML sitemap at build time
  - Includes all 15 pages (homepage, 6 main pages, 8 representation pages)
  - Sitemap available at `https://finan.eu.com/sitemap-index.xml`
  - Configured with site URL in `astro.config.mjs`

- **Installed `astro-seo` package**
  - Comprehensive SEO component for meta tags
  - Supports Open Graph, Twitter Cards, and standard meta tags
  - Integrated into main Layout component

- **Created centralized SEO configuration (`src/data/seo.config.ts`)**
  - **Basic SEO**: Site title, description, URL, author, language
  - **Title templating**: Automatic formatting with `%s | FiNAN` pattern
  - **Open Graph**: Social media sharing with image dimensions (1200x630)
  - **Twitter Card**: Twitter-specific sharing configuration
  - **Meta tags**: Keywords, robots, theme color, mobile settings
  - **Helper functions**: `getFullUrl()`, `getPageTitle()`, `mergeSEOConfig()`
  - Full TypeScript type safety with interfaces

- **SEO Usage Documentation (`SEO-USAGE.md`)**
  - Complete guide for configuring SEO settings
  - Page-level SEO prop examples
  - Social sharing image guidelines
  - Testing tools and troubleshooting tips

### Meta Tags Generated

The new SEO system automatically generates:
- Title with template support
- Canonical URLs
- Description meta tags
- Open Graph tags (11 total) with image dimensions and alt text
- Twitter Card tags (7 total) with creator and site handles
- Author and keywords meta tags
- Theme color for browser UI (`#1e3a8a`)
- Mobile web app meta tags
- Apple mobile web app status bar styling

## Changed

### Layout Updates

- **Refactored `src/layouts/Layout.astro`**
  - Replaced manual meta tags with `astro-seo` component
  - Added Props interface with JSDoc comments for better DX
  - Integrated `seoConfig` from centralized configuration
  - Support for page-level SEO overrides:
    - Custom titles (with automatic template formatting)
    - Custom descriptions
    - Custom social sharing images
    - Article metadata (publishedTime, modifiedTime)
    - Additional keywords (merged with defaults)
    - Indexing control (noindex, nofollow)

- **Updated `astro.config.mjs`**
  - Added `site: 'https://finan.eu.com'` for sitemap generation
  - Added `integrations: [sitemap()]` configuration

## Removed

- **Deleted manual sitemap implementation**
  - Removed `src/pages/sitemap.xml.ts` (custom TypeScript sitemap)
  - Replaced with official `@astrojs/sitemap` plugin for better maintainability

## Notes

- All SEO settings are now centralized in `src/data/seo.config.ts`
- Pages automatically inherit default SEO values from config
- Individual pages can override any SEO value via Layout props
- Social sharing images should be 1200x630px for optimal display
- Twitter handles in config should be without the `@` symbol
