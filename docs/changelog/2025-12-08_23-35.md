# Changelog - 2025-12-08

## Changed

### Asset Organization
- **Flag assets reorganization**: Moved flag SVG files from `public/flags/` to `public/assets/flags/` directory for better organization and consistency with other assets
- **Flag file naming standardization**: Renamed flag files from numbered format (e.g., `078-greenland.svg`) to standardized format (e.g., `flag_greenland.svg`) for improved maintainability

### Component Updates
- Updated `NordicRepresentation.astro` to reference new flag asset paths (`/assets/flags/flag_*.svg`)
- Updated `HeroHeader.astro` with improved hero configuration integration
- Updated `Navbar.astro`, `TopBanner.astro`, and `EventHeaderSection.astro` to use new flag asset paths
- Updated `index.astro` page to reflect component changes

### Configuration
- Updated `heroConfig.ts` with refined hero section content and button configurations

## Removed

- Removed old flag files from `public/flags/` directory (replaced by standardized versions in `public/assets/flags/`)
