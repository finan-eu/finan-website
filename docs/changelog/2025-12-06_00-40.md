# Changelog - 2025-12-06 (Update)

## Fixed

### Ghost API Filter Implementation
- **Fixed Ghost API filter syntax for ts-ghost library** (src/lib/ghost.ts)
  - Changed from `filter: "tag:${tagSlug}"` to `filter: "tags:${tagSlug}"` (plural)
  - ts-ghost library requires "tags:" (plural) field name, different from standard Ghost API which uses "tag:" (singular)
  - Ghost API server-side filtering now works correctly - no more fallback to client-side filtering
  - Verified with Playwright testing on Finland and Sweden representation pages

## Performance Improvements

### Blog Post Filtering
- **Ghost API now performs server-side filtering** instead of client-side
  - Previously fetched ALL posts and filtered in JavaScript (inefficient)
  - Now uses Ghost API's native filtering capability (efficient and scalable)
  - Significantly reduces bandwidth usage and improves page load times
  - Only relevant posts are fetched from the API
