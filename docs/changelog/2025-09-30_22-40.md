# Changelog - 2025-09-30 22:40 (Europe/Helsinki)

## Font Update: Switched to Open Sans

### Overview

Implemented Open Sans font across the entire website to provide a cleaner, more modern appearance as requested. This change replaces the previous Inter and Inter Tight font family with Open Sans for both body text and headings.

### Changes Made

#### 1. Font Import Update (`src/layouts/Layout.astro`)

- Replaced Inter and Inter Tight Google Fonts import with Open Sans
- Updated font weights: 400, 500, 600, 700, 800
- Maintained optimized preload strategy for performance
- Updated both regular and noscript font loading tags

**Before:**

```html
href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800&family=Inter+Tight:wght@700&display=swap"
```

**After:**

```html
href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@400;500;600;700;800&display=swap"
```

#### 2. CSS Font Family Update (`src/styles/global.css`)

- Updated `html` element font-family from Inter to Open Sans
- Updated heading elements (h1-h6) font-family from Inter Tight to Open Sans
- Maintained fallback fonts for system compatibility

**Changes:**

- Body text: `'Inter'` → `'Open Sans'`
- Headings: `'Inter Tight', 'Inter'` → `'Open Sans'`
