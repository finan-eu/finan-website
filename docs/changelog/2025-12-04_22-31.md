# Changelog - 2025-12-04 22:31 (Europe/Helsinki)

## Changed

### Standardized Domain Configuration to Use www Subdomain

Updated all configuration files to consistently use `www.finan.eu.com` as the canonical domain instead of the non-www variant `finan.eu.com`. This ensures:

- **SEO Consistency**: All generated URLs (sitemap, canonical links, Open Graph tags) now use the same domain format
- **Proper Sitemap Indexing**: Google Search Console can now correctly crawl `https://www.finan.eu.com/sitemap-index.xml`
- **Robots.txt Alignment**: The dynamic robots.txt now references the correct sitemap URL with www subdomain

#### Files Modified:

1. **astro.config.mjs**
   - Updated `site` configuration from `https://finan.eu.com` to `https://www.finan.eu.com`
   - Ensures sitemap integration generates URLs with www prefix

2. **src/data/siteConfig.ts**
   - Updated `metadata.url` from `https://finan.eu.com` to `https://www.finan.eu.com`
   - Affects canonical URLs and social media meta tags across all pages

3. **src/pages/robots.txt.ts**
   - Updated `CUSTOM_DOMAIN` constant from `finan.eu.com` to `www.finan.eu.com`
   - Ensures robots.txt sitemap reference points to `https://www.finan.eu.com/sitemap-index.xml`

#### Impact:

- All sitemap entries will now use `www.finan.eu.com` URLs
- Canonical links in page headers will use `www.finan.eu.com`
- Open Graph and Twitter Card meta tags will reference `www.finan.eu.com`
- Search engines will recognize the consistent canonical domain
