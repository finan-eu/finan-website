# Changelog - 2025-10-05 18:30

## SEO & Discoverability Enhancements

### Added

- **robots.txt** - Created robots.txt file with crawler permissions and sitemap reference
  - Allows all crawlers with crawl delay of 1 second
  - Points to sitemap at https://finan.eu.com/sitemap.xml

- **Dynamic XML Sitemap** - Implemented automated sitemap generation at `/sitemap.xml`
  - Auto-discovers all static pages and chapter pages
  - Includes last modified dates based on file modification times
  - Configurable change frequencies and priorities per page type
  - Proper XML formatting with UTF-8 encoding

### Changed

- **URL Standardization** - Renamed FAQ page route from `/faqs` to `/faq`
  - Updated `src/pages/faqs.astro` → `src/pages/faq.astro`
  - Updated navigation links in `src/components/Navbar.astro` to point to `/faq`
  - Updated sitemap configuration to reflect new `/faq` route
  - Ensures consistent URL structure across the site

## Impact

These changes improve the website's search engine visibility and discoverability by:

- Providing clear crawler guidelines via robots.txt
- Automatically maintaining an up-to-date sitemap for search engines
- Standardizing URLs for better SEO consistency
