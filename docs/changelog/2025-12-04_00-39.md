## Added

### Structured Data (JSON-LD) for SEO Enhancement

- **Created `src/data/structuredData.config.ts`**
  - Type-safe configuration for Schema.org structured data
  - `generateOrganizationSchema()` function for Organization schema
  - `generateWebSiteSchema()` function for WebSite schema
  - Integrates with existing `seo.config.ts` and `socialMediaConfig.ts`
  - Follows project pattern with TypeScript interfaces and `as const satisfies`

- **Created `src/components/StructuredData.astro`**
  - Reusable component for injecting JSON-LD into page head
  - Accepts `schemas` prop to specify which schemas to include
  - Renders `<script type="application/ld+json">` tags
  - Can be used on any page for future enhancements

- **Updated `src/pages/index.astro`**
  - Added StructuredData component to home page via head slot
  - Includes both Organization and WebSite schemas
  - Helps Google identify site name as "FiNAN" instead of "EU.COM"
