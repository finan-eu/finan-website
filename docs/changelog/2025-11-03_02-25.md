# Changelog - 2025-11-03 02:25 (Europe/Helsinki)

## Added

### FAQ Page Accessibility Enhancements

- **Skip-to-content link**: Added keyboard-accessible skip navigation link that allows users to bypass the header and jump directly to FAQ content
  - Visually hidden by default using `sr-only` utility class
  - Becomes visible when focused with proper styling and focus indicators
  - Links to `#faq-content` anchor
- **ARIA labels and landmarks**: Comprehensive ARIA attributes for improved screen reader support
  - Section-level `aria-labelledby` pointing to descriptive heading
  - List-level `role="list"` and `aria-label` attributes
  - Individual FAQ items with `role="listitem"` and unique IDs
  - Answer paragraphs linked to questions via `aria-describedby`
- **Hidden semantic heading**: Added visually hidden `<h2>` element ("List of Frequently Asked Questions") for screen readers to maintain proper document outline

## Fixed

### FAQ Page Heading Hierarchy

- **Corrected heading levels**: Changed FAQ question headings from `<h2>` to `<h3>` to fix heading hierarchy issues
  - Previously jumped from h1 (PageHeader) directly to h2 inside list items
  - Now follows proper semantic structure: h1 → h2 → h3
  - Improves accessibility and SEO by maintaining logical document outline

## Technical Details

- All changes made to `src/pages/faq.astro`
- Code formatted and linted according to project standards using ESLint and Prettier
- Tailwind CSS classes automatically sorted for consistency
- WCAG 2.1 compliance improved for keyboard navigation and screen reader support
