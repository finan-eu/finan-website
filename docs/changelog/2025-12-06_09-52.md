# Changelog - 2025-12-06 09:52 (Europe/Helsinki)

## Changed

### Sweden Working Committee Image Handling Refactored

- **File**: `src/data/representation/working-committee/swedenCommittee.ts`
- **Description**: Refactored image handling to use ES module imports instead of string paths, aligning with the pattern used in Finland committee data.

**Changes made:**
- Added 6 image imports at the top of the file for all committee member photos
- Replaced all `imageSrc` string paths with imported image variables
- Ensures consistency with `finlandCommittee.ts` implementation

**Benefits:**
- Enables Astro's build-time image optimization (compression, format conversion, responsive variants)
- Provides TypeScript type safety and build-time validation for image paths
- Implements asset fingerprinting for better browser caching
- Allows the bundler to track image dependencies and tree-shake unused assets
- Prevents silent failures from broken image paths

This change improves code quality, performance, and maintainability across the working committee data layer.
