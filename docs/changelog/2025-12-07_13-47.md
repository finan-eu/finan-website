# Changelog - December 7, 2025

## Changed

- **Replaced `@astrojs/sitemap` plugin with manual sitemap implementation**
  - Removed `@astrojs/sitemap` integration from `astro.config.mjs`
  - Created type-safe sitemap configuration using TypeScript
  - Implemented manual XML sitemap generator at `src/pages/sitemap.xml.ts`
  - Added comprehensive SEO metadata for all pages:
    - Priority values (1.0 for homepage, 0.8-0.4 for other pages)
    - Change frequency indicators (weekly, monthly, yearly)
    - Last modified timestamp for homepage
  - Updated `robots.txt` to reference new sitemap location (`/sitemap.xml`)

## Added

- **New sitemap configuration files**
  - `src/data/sitemap/types.ts` - TypeScript type definitions for sitemap structure
  - `src/data/sitemap/sitemapConfig.ts` - Centralized sitemap data with SEO priorities
  - All 16 site pages included with appropriate SEO metadata

## Improved

- **Better SEO control**
  - Granular priority settings per page
  - Proper change frequency hints for search engines
  - XML escaping for security
  - Correct Content-Type headers (`application/xml; charset=utf-8`)
  - Cache-Control headers for optimal performance

## Technical Details

The manual sitemap implementation provides:
- Full control over individual page SEO metadata
- Type-safe configuration following project patterns
- Easy maintenance when adding new pages
- Better debugging capabilities
- Consistent with existing data architecture (`as const satisfies` pattern)

This change addresses potential indexing issues with Google Search Console and provides more control over how search engines crawl and index the site.
